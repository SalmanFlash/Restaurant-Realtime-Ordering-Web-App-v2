# 🍽️ Restaurant Ordering System v2

A real-time QR-code based ordering system for restaurants.

## How it works
- **Customer:** Scans table QR code → browses menu → places order
- **Kitchen:** Real-time dashboard shows incoming orders → mark as ready

## Tech Stack
- Vanilla HTML, CSS, JavaScript
- Bootstrap 5 (responsive design)
- Firebase Firestore (real-time database)
- Firebase Hosting (deployment)

## Features
- Multi-restaurant support via URL parameters
- Real-time order sync (no refresh needed)
- Kitchen display with order status management
- Mobile-first customer interface
